## introducción

Comenzamos hablando del examen.

¿qué cuenta como relato?

cuantos, teatro, historia, sucesos, hacer un completo, sueños, un código de p5.js, haiku, memes, chistes, etc.

¿qué no contaría como relato?

una descripción de un hecho. Para que cuente como relato debe tener algo salido de tu opinión.

El examen será un **relato interactivo en 3 pasos**

## cátedra

revisamos la sección de archivos de p5.js(a la izquierda del código)

### HTML

HyperText Markup Language: Es el archivo que carga el sketch en el navegador.
Generalmente no necesitas tocarlo, pero es útil saber que ahí es donde se
importa la librería de p5.js y donde vive el canvas.

### CSS

Cascading Style Sheets: Controla la apariencia de la página que rodea el canvas
(fondo, márgenes, etc). Tampoco sueles editarlo en p5.

### sketch.js

El archivo principal donde escribes tu código. Aquí se determinan las interacciones.

para el examen deben subir su proyecto a una página web, utilizando GitHub.
